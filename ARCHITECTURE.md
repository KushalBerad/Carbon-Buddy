# Carbon Buddy Architecture

Carbon Buddy is an AI-powered sustainability awareness platform.

## Frontend Layer

- React 19
- TypeScript
- Tailwind CSS
- Motion UI animations

## State Management

- Zustand persistent store architecture

## Backend Layer

- Express.js server

## AI Layer

- Google Gemini API integration for sustainability recommendations

## Deployment

- Render cloud deployment

## Core Features

- Carbon footprint awareness dashboard
- AI sustainability coaching
- Eco-friendly meal recommendation engine
- Earth Bloom gamification system
- Weekly sustainability reflection tracking
